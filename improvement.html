授業について

    
学習方法改善のヒント


    
学習に対する姿勢

    自己評価は非常に大切です。
    自分の実力を正しく認識することでどのように学習すればいいのかがわかります。
    自分の悪いところを改善しようということも重要ですが、自分の良いところを認識してそれを伸ばすことも非常に大切です。

    常に１００％を目指すのはよいことです。
    問題なのは１００％にならなければ次へ進まない、という姿勢です。
    最終的に１００％に早く到達することが目的なのであれば部分的にできていないところを飛ばしたほうが全体としては早くなることがあります。
    わからないところを保留することも重要です。
    できることを少しずつ増やしていくことにしましょう。

    
授業に対する姿勢

    授業は自分では突破できない疑問、もしくは自分では解決に時間のかかる疑問を解決するためにあります。
    またかなりレベルが上がれば自分では気づきにくいレベルの問題点を見つける場にもなります。
    「特に質問はありません。」は禁句です。これは勉強していないことに等しいと言えます。
    質問が無いのであれば完璧でなければいけません。
    かといって重要度が低い、難しいことを探して質問する人がいますがこれもいけません。
    基本を深く理解するための質問が最も学力が上がりやすい質問です。
    そもそも記述偏差値７５くらいまでは基礎固めだけで到達できます。
    偏差値７０未満の人が応用にばかり目がいくのは成長スピードを阻害する原因になります。
    趣味として難しい話を聞きたいのであればいいですが、学力を上げるために難しいことばかりに目がいくようでは本末転倒です。

    言葉をそのまま理解する姿勢が必要です。
    説明を聞いた時に、「つまり○○ということですか？」と確認することはとても良いことです。
    ただし、そのときに自分が解釈する時に情報を付け加えたり変えたりして解釈してしまうと誤解の原因になります。
    中身を変えずに表現を変えるのは良いことです。

    「〜になった時」という言葉には二通りの解釈があります。「その瞬間」と「なった直後（過去形）」です。
    文脈も含めて誤解しないように気をつけましょう。
    「〜になる時」には「なる瞬間」と「なるとしたら（仮定）」の解釈があります。
    わかりにくい場合は質問してください。

    説明を受けた時に自分の言葉の使い方がおかしいと誤解の原因になります。
    指摘を受けたときにはまずそのまま直接理解するように心がけます。
    疑問が出る原因が「自分は○○という言葉は△△という意味で使ってきたから」というのが邪魔をすることがあります。

    曲解は誤解の元です。素直に、直接理解する癖をつけましょう。



    
学力向上とはなにか

    学力向上とはできることが増えることです。
    スモールステップを意識します。
    解ける問題が一問増えたのならばそれは学力が上がっています。
    全部で１０００パターン解けるようになりたいのであれば解けるものを１つ増やすことを１０００回すれば到達できます。
    学力向上とは成績上昇とは異なります。
    自身の学力が向上しても周囲の学力向上の方が早ければ成績の見た目は下がります。

    
学習スピードを上げるコツ

    ポイントを聞いてすぐに要点を書き留める。
    自分の疑問を簡潔に伝えることができる。
    原因を論理的に突き止める。

    
困ったときのQ&A

    問題が解けない　
    →　問題が解けない原因は自分で把握できていますか？
    問題に対して何から取り組めば良いのかわからない。
    →　問題を解く第一歩は問題把握です。まずは問題把握を正確にしましょう。
    問題は把握できるけれども何から手を付ければいいのかわからない。
    →　思考問題であれば試行錯誤することが必要な場合もあります。答えではなく新しくわかる何かを一つ見つけましょう。
    例：浮力を求める問題でわからないけれども、物体の質量なら求められる。（一歩前進です。）

    問題が解けない理由がわからない　
    →　問題に使われている用語はすべて理解して自分で説明できるレベルになっていますか？
    問題に使われている用語は何となく雰囲気はわかる　
    →　定義や概念を正しく覚えましょう
    特定の用語に対する定義や概念がわからない　
    →　教科書で確認しましょう
    教科書を見ても定義や概念がわからない　
    →　参考書をあくまでも「参考に」見てみましょう。
    教科書よりも正確な記述のある参考書は教科書よりも難しいことがほとんどです。
    問題が把握できない
    →　問題の情報を整理整頓しましょう。箇条書きにしたり図にしたり表にしたりします。

    教科書や参考書を見ても定義や概念がわからない　
    →　どのような言葉がわからないのか具体的に文章に書き出してみましょう。
    具体的に特定の用語について定義や概念がわからない　
    →　教科書に書いてある言葉をそのまま書いて説明問題（記述）の答えとして書いてみましょう。
    わかったけれども何となく腑に落ちない　
    →　それは論理的な疑問ですか？気分的な疑問ですか？
    論理的だと思えない　
    →　どこについて論理や飛躍していると感じるのか疑問を書き出してみましょう。
    論理的には正しいと思うけれどもなんとなく腑に落ちない　
    →　それは気持ち的に受け入れられなくなっている状態です。
    簡単な例を作成して試してみたり、何度も触れることで徐々に慣れるようにしましょう。
    あまりにも詰まるようであれば保留しましょう。


    
学習がうまくいっていない人の特徴

    予定がコロコロ変わる人　→　学習に関することを最優先にすべき
    学習の予定がよく変わる人は計画的に学習を進める習慣をつけます。

    「今週は何を学習しましたか？」
    この質問に対して明確に自分が学習してきたことがすぐに出てくること。
    さらにはどんな成果が得られたのかを話せること。
    これができるようになると自分でも何をすべきかがわかっていてそれを実行する習慣がついたことになります。

    
失敗する人の特徴

    失敗する人から学ぶことが重要です。
    失敗の原因がはっきりしている人がいます。
    その失敗する人たちと同じ行動を取っている場合は改善が必要です。
    優先すべきことは学力向上や受験での成功です。

    
できる人の共通項目

    できる人は解いたらすぐに丸付けをして間違ったところの修正をする。
    「解く→丸付け→修正」が１セットで途中で区切らない。

    できる人は基本的な質問が多い。
    できる人は定義や基本的な数式の導出を意識するので自分の認識が正しいかどうかを確認するための質問が多くなる。

    できる人は質問回数が多い。
    勉強量が多いので疑問点も多く生まれる。そのため解消すべき質問が多い。

    できる人は高得点の試験についても質問する。
    ほとんど正解しているようなテストに対してもできる人は質問事項が存在する。
    完全な理解を目指すので１００点の試験であっても曖昧な理解があれば質問がある。

    できる人は正解している問題についても質問する。
    選択肢問題であればすべての選択肢に疑問がないレベルまで解消する。ただし珍しい問題や難しい問題であればその限りではない。

    できる人は共通テストやセンター試験過去問などで９０％を悪い点数だと思っている。
    全範囲ができなければならない難易度の試験において９０％ということは弱点があるということを認識することが大切。
    １００％であっても曖昧な部分があれば復習すべき。
    
    できる人は学習効率が良い行動を取る。
    学習効率の良いことが最優先であって、復習のタイミングやサイクルに慢心が無い。

    できる人は教科書のレベルの高さを知っている。
    できる人は教科書の記述内容が簡潔であるため理解に時間がかかることもあることを知っている。

    できる人は科目の中の目次が言える。
    できる人は科目ごとに学習すべき分野を認識し、全体像を把握しているので目次が言える。

    できる人は教科書を使いこなせる。
    できる人は定義や用語の確認が重要だということを認識しているので学習の際に教科書で確認をしている。

    
成長する人と失敗する人の違い

    成長する人は目的を持って学習する。
    失敗する人は目的があいまいなまま学習する。

    成長する人は短期的な目標をもっている。
    失敗する人は短期的な目標に無頓着。

    成長する人は迷う時間が極端に短い。
    失敗する人は迷うことに時間を使う。

    成長する人は教材を決めたら前に進める。
    失敗する人は教材選びに時間を使い続ける。

    成長する人は解いたらすぐに自己採点して復習する。
    失敗する人は解くだけ。

    できる人は模試の成績把握が正確。
    できない人は過去の模試の成績がどうだったか曖昧。

    できる人は試行錯誤する。
    できない人は習っていないからできないと言い訳する。

    
努力量が不足している人の特徴

    宿題をしない。
    宿題よりも自分が必要と思った学習をする。（宿題をした上で自分の学習もすべき）
    宿題を見た目だけの仕上げにしている。（中身がない）

    
勉強しているのに失敗する人の特徴

    何を勉強しているのかを自覚していない。
    周りから色々と言われすぎて自分でどうすればいいのか決断できていない。
    周囲のあいまいな助言と活用すべき助言の区別ができていない。
    周囲に振り回される。
    自分に適した問題集、教材をするのではなく周りが使っているものと同じものを使う。
    基本を疎かにしている。
    目標があいまい。
    面倒なことから逃げている。
    時間に対する感覚がルーズ。
    計画性が無い。
    受け身で自分から動かない。
    正しいことを感情面から受け入れられない。
    論理を優先していない。
    感情面を優先している。
    自分の能力と実際に取り組む際のレベルが釣り合っていない。
    無理なことに挑戦している。
    簡単すぎることばかりしている。
    基本をおろそかにしている。
    基本を身につけるために何をすべきかがわかっていない。
    助言を活用できない。
    成長することよりも楽をすることを優先している。
    先入観が邪魔をして改善案を受け入れられない。
    判断力が身についていないにも関わらず自分には無理だという判断は正しいと勘違いしている。
    多くの人がする失敗と同じレールに乗っていることを指摘されても改善しない。
    目先の定期テストや確認テストでの結果にこだわって一時的な学習になってしまい長期的な学習ができていない。
