mac vscode ショートカットキー

    cmd + O 開く
    cmd + N 新規ファイル作成
    ctrl + P カーソルを上に移動
    ctrl + N カーソルを下に移動
    ctrl + F カーソルを右に移動
    ctrl + B カーソルを左に移動
    shift + cmd + K カーソル行削除
    cmd + enter 下に行追加
    option + ↓ カーソル行を下に移動
    option + ↑ カーソル行を上に移動
    shift + option ↓ カーソル行を下にコピー
    shift + option ↑ カーソル行を上にコピー
    option + cmd + F 置換
    cmd + F 検索
